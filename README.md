# Li Sining Academic Portfolio

This repository contains a GitHub Pages personal academic homepage for Li Sining / 李司宁.

## Files

- `index.html` contains the page content and section structure.
- `style.css` controls the academic portfolio visual design and mobile layout.
- `script.js` controls the mobile navigation menu and footer year.
- `README.md` explains how to maintain the site.

## Updating Website Content

Most updates can be made directly in `index.html`.

Common edits:

- Update the profile summary in the `Home` and `About` sections.
- Add or revise degree information in the `Education` section.
- Adjust research themes in `Research Interests`.
- Replace project descriptions in `Projects`.
- Add writing topics, reading notes, or essay links in `Writing & Notes`.
- Replace `Add your email address here` in `Contact` with a real email address.

## Publishing With GitHub Pages

1. Commit these files to the repository.
2. Open the repository on GitHub.
3. Go to `Settings` > `Pages`.
4. Under `Build and deployment`, choose `Deploy from a branch`.
5. Select the branch that contains this site, usually `main`, and choose the root folder.
6. Save the settings and wait for GitHub Pages to publish the website.

## Content Guidelines

Keep the site factual and academic. Add publications, awards, supervisors, or research outputs only after they exist and can be accurately described.
