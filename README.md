# AI & Education Research Portfolio

This repository contains a minimal GitHub Pages research portfolio designed for Education PhD applications.

## Research Identity

AI & Education Researcher studying learning systems and student autonomy in AI-enhanced environments.

## Site Structure

- `index.html` contains the semantic page structure.
- `style.css` defines the minimal academic visual system and responsive layout.
- `script.js` stores portfolio content in modular data objects for easier future updates.
- `README.md` documents the website purpose and maintenance model.
